I'm building a research agent on top of Claude Code that produces validated summaries of AI/ML topics — primarily alignment, post-training, evaluation, and interpretability. This site is where the validated output goes.

Each note follows the same structure: definition, mechanism, lineage, state of the art, key papers, open questions, sources. The notes interlink as a wiki; the open-questions page aggregates the unsettled questions surfaced across the corpus.

If you find a note useful, broken, or wrong, I'd like to know.

- **[Open questions](/open-questions)** — what the corpus says is still unsettled, organised by topic
- **[Method](/method)** — how the research pipeline works, what it does well, where it fails
